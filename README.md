## 1)  Instalar python y npm

https://www.python.org/downloads/
https://nodejs.org/en/

## 2) Instalar las dependencias necesarias para ejecutar nuestro proyecto
`/flaskr/`
```bash
pip install -r requirements.txt
```

`/front/`
```bash
npm install
```

## 3) Ejecutar el frontend y el backend

`/flaskr/`
```bash
py ./main.py
```

`/front/`
```bash
npm run dev
```
